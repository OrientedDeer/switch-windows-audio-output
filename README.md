# Swap Left and Right Speakers on Windows

Windows has no setting to swap stereo channels. The balance slider doesn't work — it mutes one side instead of swapping. Here's what actually works.

## Download

**[Download SwapAudio.exe](https://github.com/OrientedDeer/switch-windows-audio-output/releases/latest/download/SwapAudio.exe)** — single file, no dependencies.

- Run once to swap L/R channels
- Run again to swap back
- `SwapAudio.exe uninstall` to remove completely

Requires administrator privileges. Installs a lightweight audio processing driver on first run.
